Klocwork Server

Provide the link to the Klocwork Server that you want to connect with. The name is used as an identifier and is referenced on a job configuration under Build Environment. The URL should be in the form: http://kwserver:8080

Specific license settings

Organisations can have multiple Klocwork Servers but a single license server. Provide the license settings under the Global License Settings.

If you use different license servers, provide the specific license settings that you want to use, depending on which Klocwork Server URL you are using. Select the "Use specific License Settings for this Klocwork Server" check box and specify the license host and port.
