Klocwork サーバー

接続したい Klocwork サーバーへのリンクを指定します。名前は識別子として使用されるほか、[ビルド環境] のジョブ設定で参照されます。URL は、http://kwserver:8080 という形式にする必要があります。

具体的なライセンス設定

Organisations can have multiple Klocwork Servers but a single license server. Provide the license settings under the Global License Settings._JA

If you use different license servers, provide the specific license settings that you want to use, depending on which Klocwork Server URL you are using. Select the "Use specific License Settings for this Klocwork Server" check box and specify the host and port._JA
