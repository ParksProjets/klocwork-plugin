The Klocwork tool used to generate the build specification, by default this is set to kwinject.
Tools Available:
kwinject monitors your C/C++ build to create a build specification
kwmaven creates a Java build specification based on the information in your maven build file
kwmavenw creates a Java build specification based on the information in your maven wrapper file
kwjsspec creates a Javascript build specification file
kwpyspec creates a Python build specification file
kwktspec creates a Kotlin build specification file
kwant creates a Java build specification based on the information in your ant build file
create tracefile uses kwinject to create a trace file, mainly used for debugging
kwgradle creates a Java build specification based on the information in your Gradle build file
kwgradlew creates a Java build specification based on the information in your Gradle wrapper file
