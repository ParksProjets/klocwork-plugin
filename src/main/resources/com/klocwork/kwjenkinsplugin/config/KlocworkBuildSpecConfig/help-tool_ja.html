build specification (ビルドスペック) の生成に使用される Klocwork ツール。デフォルトでは、kwinject に設定されています。
利用可能なツール:
kwinject C/C++ ビルドを監視して、build specification (ビルドスペック) を作成します
kwmaven maven ビルドファイル内の情報に基づいて、Java build specification (ビルドスペック) を作成します
kwmavenw maven ラッパーファイル内の情報に基づいて、Java build specification (ビルド スペック) を作成します
kwjsspec JavaScriptビルドスペックファイルを作成します
kwpyspec Pythonビルドスペックファイルを作成します
kwant ant ビルドファイル内の情報に基づいて、Java build specification (ビルド スペック) を作成します
kwvcprojparser kwinject を使用して Microsoft Visual Studio 2012 または 2015 C/C++ プロジェクトの build specification (ビルドスペック) を作成できない場合の代替策となります。ただし、このコマンドは前世代 (Klocwork 2018 より前) の解析エンジンに基づいて結果を出すだけです
kwcsprojparser Visual Studio C# プロジェクトとソリューション向けに build specification (ビルド スペック) を作成します
create tracefile kwinject を使用して、トレースファイル (主にデバッグに使用される) を作成します
kwgradle Gradle ビルドファイル内の情報に基づいて、Java build specification (ビルド スペック) を作成します
kwgradlew Gradle ラッパーファイル内の情報に基づいて、Java build specification (ビルド スペック) を作成します
